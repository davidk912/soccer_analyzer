기획

어떤 문제를 왜 해결하려 했는가

🎯 문제 인식

아마추어 축구팀은 프로팀과 달리 전담 분석가나 고가의 분석 장비가 없습니다. 드론 촬영이 보편화되면서 팀의 경기를 위에서 찍는 것은 쉬워졌지만, 그 영상을 전술적으로 분석하는 도구는 여전히 전문가 영역에 머물러 있습니다.

💡 해결 아이디어

드론 탑뷰 영상을 AI에 입력하면 선수 위치를 자동으로 추출하고, 진형을 감지하여 상대 진형에 맞는 전술 조언을 한국어로 제공하는 프로그램을 제작합니다. 누구나 설치하여 바로 사용할 수 있는 Windows 데스크톱 앱이 목표입니다.

🏆 기대 효과

경기 후 5분 이내에 객관적 전술 분석 리포트를 얻을 수 있습니다. 히트맵과 이동 경로로 특정 선수의 활동량과 포지션 유지 여부를 직관적으로 파악하고, 다음 경기 전략 수립에 바로 활용할 수 있습니다.

구성물

주요 기능 목록

  • 🎬

    영상 입력 (로컬 / YouTube)

    mp4·avi·mov 등 로컬 파일과 YouTube URL 모두 지원합니다. YouTube는 yt-dlp로 스트리밍 URL을 추출하여 다운로드 없이 바로 분석합니다.

  • 🤖

    AI 선수 자동 감지

    YOLOv8n(nano) 모델이 드론 영상에서 선수를 실시간으로 감지합니다. 샘플 간격을 조절하여 분석 속도와 정확도의 균형을 맞출 수 있습니다.

  • 👕

    팀 자동 분리

    유니폼 상단 영역의 HSV 색상을 추출하고 KMeans(k=2)로 두 팀을 자동 구분합니다. 동일 색상일 경우 필드 좌/우 위치로 대체 분리합니다.

  • 📐

    진형 분석 (6종)

    KMeans 클러스터링으로 선수 군집을 형성하고, 미리 정의된 6가지 진형 템플릿(4-4-2, 4-3-3, 3-5-2, 4-2-3-1, 5-3-2, 4-1-4-1)에 최근접 매칭합니다.

  • 💬

    한국어 전술 추천

    두 팀의 감지된 진형 조합에 따라 구체적인 한국어 전술 가이드를 즉시 제공합니다. 측면 공략, 압박 타이밍, 세트피스 등 실전 조언을 포함합니다.

  • 🌡️

    히트맵 시각화

    팀 A / 팀 B / 전체를 선택하여 선수 활동 밀도를 축구 필드 위에 색상 밀도 지도로 표현합니다. 어느 구역에서 많이 활동했는지 한눈에 파악 가능합니다.

  • 🗺️

    이동 경로 추적

    경기 전반에 걸친 선수들의 이동 경로를 팀별로 필드 위에 표시합니다. 팀 A·B 각각 토글 가능합니다.

🤖 AI 도구(Claude Code) 활용 과정

계획 수립 (Plan Mode)

Claude Code의 계획 모드를 통해 1시간 내 구현 가능한 범위를 설정하고, 파일 구조·빌드 순서·위험 요소를 체계적으로 정리했습니다.

핵심 로직 코드 생성

analyzer.py(YOLO 감지, KMeans 진형 분석, 전술 추천 테이블), visualizer.py(matplotlib→QPixmap 변환, 필드 배경 드로잉)를 자동 생성했습니다.

GitHub 구조 재편성 및 문서화

대회 규정(src/, Release/, README.md)에 맞게 프로젝트 구조를 재정비하고 build.bat, .gitignore, 보고서까지 자동 작성했습니다.

특장점

이 프로그램만의 강점

📶 YouTube 직접 분석

URL만 붙여넣으면 다운로드 없이 YouTube 영상을 스트리밍으로 바로 분석합니다. 외부 도구가 필요 없습니다.

🇰🇷 한국어 전술 조언

진형 조합 6×6 = 36가지 시나리오에 맞춘 실전 한국어 전술 가이드를 즉시 제공합니다.

⚡ 설치 즉시 사용

YOLOv8 모델이 첫 실행 시 자동 다운로드됩니다. 별도 설정 없이 pip install 후 바로 실행 가능합니다.

🧵 UI 프리징 없음

QThread 기반 비동기 워커로 AI 분석 중에도 UI가 완전히 반응합니다. 진행 바로 분석 상태를 실시간 확인합니다.

🎨 직관적 다크 UI

GitHub 스타일 다크테마로 설계된 UI로 피로감 없이 분석 결과를 확인할 수 있습니다.

🔧 속도 조절 가능

샘플 간격 슬라이더로 분석 정확도와 속도를 사용자가 직접 조절할 수 있습니다.

한계점

현재 버전의 한계

✓ 잘 되는 것

  • 탑뷰 드론 영상에서 선수 감지
  • 유니폼 색상이 뚜렷이 다른 두 팀 분리
  • 팀별 활동 영역 히트맵 시각화
  • YouTube 스트리밍 영상 분석
  • 한국어 전술 텍스트 즉시 출력

✗ 현재 한계

  • 유니폼 색상 유사 시 팀 분리 오류 가능
  • 개인 선수 식별 불가 (번호·얼굴 미지원)
  • 공 위치 추적 미지원
  • 사이드뷰·저각도 영상 분석 불가
  • 관중·심판이 선수로 오감지 될 수 있음
  • GPU 없이 느린 분석 속도 (긴 영상 시 수분 소요)

Todo

향후 개선 방향

HIGH

개인 선수 ID 추적 (Re-ID)

딥러닝 기반 사람 재식별(Re-Identification) 모델을 도입하여 같은 선수를 프레임 간 지속 추적하고 개인별 이동 거리·히트맵을 생성합니다.

HIGH

공 추적 및 패스 맵

YOLO sports ball 클래스로 공 위치를 추적하고, 어느 선수 간에 패스가 이뤄졌는지 패스 네트워크 그래프로 시각화합니다.

MID

GPU 가속 지원

CUDA 환경 자동 감지 후 GPU 추론으로 전환하여 분석 속도를 10배 이상 향상시킵니다. 긴 경기 영상도 수십 초 이내 분석 가능하게 합니다.

MID

분석 결과 PDF 내보내기

히트맵, 진형 다이어그램, 전술 추천을 포함한 경기 분석 보고서를 PDF로 저장하여 팀원과 공유할 수 있게 합니다.

LOW

다국어 전술 추천

한국어 외 영어·스페인어·포르투갈어 전술 추천을 추가하여 해외 아마추어 팀도 사용할 수 있게 합니다.

LOW

경기 영상 구간 클리핑

분석 결과에서 특정 전술 패턴이 발생한 타임스탬프를 자동 감지하고 해당 구간만 잘라서 저장하는 기능을 추가합니다.
